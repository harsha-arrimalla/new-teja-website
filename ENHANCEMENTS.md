## 🛠️ Customization

### To Change WhatsApp Number:
Search for `919391008801` and replace with your number in:
- `index.html`
- `shop.html`

### To Adjust Discount Percentages:
In the JavaScript sections, modify:
```javascript
const discountPercent = Math.floor(Math.random() * 40) + 20; // 20-60% off
```

### To Change Color Scheme:
Edit CSS variables in `style.css`:
```css
--gold-accent: #B8860B;
--red-accent: #862B2B;
```
